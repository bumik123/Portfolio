Selenium

-Selenium is an automation tool developed for only webbased application.
-Selenium was developed by jason in year of 2004.
-Selenium is open source and free automation software.
-Selenium supports upto 14 programming Langauges or language binding.
-Selenium supports all the browser.
-Majorly used selenium is 'Selenium WebDriver'.
